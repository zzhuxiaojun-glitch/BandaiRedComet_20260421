# 万代抢购器 · 使用指南

**版本**：v0.2.0
**支持平台**：Windows 10 / 11

---

## 第一次用（5 分钟）

### 1. 解压 & 启动

把 `万代抢购器.zip` 解压到任意目录（建议 `C:\Bandai\`，**不要放桌面或路径含中文的目录**），然后**双击** `万代抢购器\万代抢购器.exe`。

启动 < 1 秒，弹出红色顶栏的窗口。

> **Windows Defender 警告**：第一次启动可能弹"是否允许"。点"允许 / 信任"即可。我们不做坏事，只是没花钱买代码签名证书。

### 2. 拿一个 CK（你的"通行证"）

CK 是登录万代小程序后服务器发的身份凭证（一串 175 字符的字符串）。**每次抢购前都要保证 CK 还有效**（一般 JWT 几天会过期）。

**有现成的 HAR 文件**（`*.har`）：
- 点 GUI ① 账号卡片下的 **「📁 从 HAR 导入 CK」** 链接
- 弹出文件选择器 → 选你的 .har 文件
- CK 自动填入 + 自动验证（看到黄色 ✅ 的提示就 OK）

**没有 HAR 文件**：
- 让我（脚本作者）给你抓一份发给你 / 教你怎么抓
- 或者用我之前给你的「上号软件」抓 Token 工具

CK **看着像乱码就对了**，长这样：
```
eyJ0eXAiOiJKV1QiLCJhbGciOi...（175 字符）...g1sUpE
```

### 3. 找到要抢的商品

GUI **② 商品** 卡片顶部有搜索框：

```
关键词搜索（推荐）
[里歇尔                     ] [🔍 搜索]
```

输入商品名（如 "里歇尔"、"高达"、"假面骑士"）→ 回车 → 弹窗显示匹配的商品列表。**点一条**自动填进去，SKU 也会自动加载。

### 4. 设抢购时间 + 开始

- **③ 收货**：点过"先预检"后地址会自动出现。多个地址选默认那个就行
- **④ 开抢时间**：日期 + 时分，**填万代官方公布的开抢时间**
- 点 🔥 **开始抢购** 红色按钮

### 5. 倒计时 + 抢中

倒计时跑到 0 时 GUI 会自动尝试下单，**1-2 秒内**显示结果：
- ✅ 抢中 → 大字 "抢 到 了 🎉"，订单号 + prepay_id 一键复制
- ❌ 没抢到 → 失败诊断卡片，告诉你为什么（CK 失效 / 库存不足 / 风控等）

### 6. 抢中后付款

GUI 不能自动付款（微信安全限制），你得**手动**：
1. 点 GUI 里 **「打开微信」** 按钮（或自己开微信）
2. 微信里搜 **万代南梦宫官方微商城** 小程序
3. 进去 → **我的订单** → 找到刚抢的那笔（"待支付"状态）
4. 点 **去支付** → 走微信支付流程

**⚠️ 必须 15 分钟内付款**，否则订单自动取消，库存释放给下一个人。

## 平时怎么抢（每次的步骤）

下次再用就更简单：

1. 双击 exe → CK 自动验证（如果之前勾了"记住 CK"）
2. CK 失效 ⚠️ 黄条提示 → 点"📁 从 HAR 导入 CK"换一份
3. 搜商品 → 选 SKU → 设时间 → 开抢
4. 抢中 → 微信付款

## 想收推送通知（抢中那一刻收到提醒）

GUI 折叠卡片 **⑥ 推送通知** 里：

| 通道 | 收推送的方式 | 怎么注册 |
|---|---|---|
| **Server 酱** ⭐ | **微信"服务通知"** | https://sct.ftqq.com/ GitHub 登录 + 微信扫码 |
| **PushPlus** | 微信公众号 / QQ / 钉钉 | https://www.pushplus.plus/ 微信扫码 |
| Bark | iPhone 推送 | App Store 装 Bark 复制 URL |
| 飞书机器人 | 飞书群 | 群机器人 webhook |

最方便：**Server 酱**。注册完拿 SCTKEY 粘到 GUI → 点"试发" → 微信里收到 = 配好了。

## 出问题怎么办

| 现象 | 怎么处理 |
|---|---|
| 双击 exe 没反应 | 检查 Windows Defender 是否拦截，加信任 |
| 启动后 ① 账号 显示 🔑 CK 已失效 | 点"📁 从 HAR 导入 CK"换最新的 |
| 搜索框打不出中文 | 切回输入法（中文/拼音）再试 |
| 抢购失败 红色卡片显示"被风控" | 等 5-10 分钟换 CK 再试，避免短时间多次抢同个商品 |
| 抢购失败 "尚未开售" | 检查抢购时间是不是设对了（万代用北京时间） |
| 抢购失败 "库存不足" | 抢晚了，没办法 |
| GUI 乱码 / 字体怪 | Windows 是不是非 10/11？升级 |

更深入的故障，把 `万代抢购器/logs/` 文件夹里**最新的 .log 文件**发给我，我看一眼就知道。

## 不能做什么

- ❌ **自动付款** —— 微信小程序支付要在小程序里完成，脚本碰不到
- ❌ **抓 CK 全自动** —— 微信沙箱限制，必须用工具或抓包获取
- ❌ **多个商品同时抢** —— 一次只抢一个 SPU
- ❌ **保证抢中** —— 万代风控 / 库存 / 网络延迟都可能导致失败

## 隐私

- CK / 订单 / 搜索历史**全部存在你本地** `%APPDATA%\BandaiSniper\`
- 软件**不联网上传任何东西**到我或者第三方服务器
- 唯一外部请求：直接打万代服务器（`crm-app-api.bandainamcoshanghai.com`）和你配的推送服务（如 sct.ftqq.com）
