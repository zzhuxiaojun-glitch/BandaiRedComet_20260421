# 万代抢购器 · 功能介绍

**一句话**：双击 exe 就能用的万代南梦宫上海微商城（PB 限定商品）自动抢购工具。

---

## 它能做的

### 🔍 找商品

- **关键词搜索**：输商品名（如"里歇尔"、"假面骑士极狐"），实时调万代搜索接口，秒出结果
- **从 HAR 选**：如果你抓过 HAR 文件，可以直接从里面挑商品
- **手动填 SPU ID**：知道编号直接填

### 🔑 账号管理

- **CK 自动验证**：启动时自动检查你的会话凭证还有没有效
  - ✅ 黄色徽章 = 有效，看到 `memberId=XXXXXX`
  - 🔑 粉色徽章 = 失效，提示你换一份
  - 🔄 蓝色 = 正在验证
- **从 HAR 一键导入 CK**：选个 HAR 文件，自动抠出最新的 CK 填进去

### 🎯 抢购流程

- **倒计时**：大字显示 `T-00:01:23`，进度条同步
- **预热期库存轮询**（瞬爆款优化）：
  - 抢购前 60 秒每 150ms 拉一次商品状态
  - 一看到库存放出 / 状态变可售 → **可比官方时间最早提前 2 秒触发**
  - 对"零秒抢光"的爆款关键
- **3 个并发请求**：同时打 confirmOrder + createOrder，谁成功谁赢
- **抢中后**：金黄色 "抢 到 了"，显示订单号 + 微信支付参数 + 用时
- **失败诊断**：14 类常见错误自动分类，告诉你具体原因 + 怎么改

### 📦 历史记录

GUI 顶部 **「📦 我的订单」** 按钮：
- 列出所有抢中的订单（含已自动取消的演练订单）
- 一键复制订单号 / 微信支付 prepay_id
- 状态徽章（待支付 / 已付款 / 已取消）

GUI 顶部 **「🔍 搜索历史」** 按钮：
- 你搜过的关键词列表，按搜索次数 + 最后时间排序
- 点一条**重新搜**，免得每次重新打字
- 一键清空

### 🔔 推送通知

抢中 / 失败那一刻给你手机推一条，4 个通道任选：

- **Server 酱** → 微信公众号"服务通知"⭐ 推荐
- **PushPlus** → 微信 + QQ + 钉钉等多通道
- **Bark** → iOS 系统推送
- **飞书机器人** → 飞书群

### ⏰ 时间智能

- **本地时区自动适配**：你在中国用就是北京时间，在外地用会显示警告并提示与北京的时差
- **过去时间禁选**：日历里今天之前的日期会灰
- **时刻自动校正**：不小心选了过去的时刻 → 自动跳到当前 +1 分钟
- **实时显示当前时间**：方便你和系统时钟对照

### ⚙️ 高级策略（默认够用）

折叠在 ⑤ 里，可调：
- 并发数（默认 3，开高了反而触发风控）
- 重试次数（默认 8）
- 提前预热秒数（默认 60）
- 最早提前 fire 毫秒（默认 2000）
- 价格护栏（结算价超过此值弃单，防钓鱼）

## 它不能做的

| 不能做的事 | 为什么 |
|---|---|
| ❌ 自动付款 | 微信小程序支付必须在小程序里走，外部脚本碰不到 |
| ❌ 全自动获取 CK | 微信对小程序 storage 沙箱化，外部进程读不到 |
| ❌ 多个商品同时抢 | 当前一次只抢一个 SPU；下版本可能加候补队列 |
| ❌ 保证抢中 | 万代风控 / 库存 / 网络延迟都可能让你抢不到 |

## 它在干什么（技术细节，可不看）

万代小程序所有 API 请求都做了 AES-192 加密 + HMAC 签名。这个软件做的事：

1. 用解包出来的密钥**模拟小程序的 API 调用**
2. 在你设的开抢时间，并发发送 `confirmOrder` + `createOrder` 请求
3. 拿到服务器返回的微信支付 prepay_id 时即抢中
4. 整个链路从触发到拿订单号通常 **150 ms - 2 秒**

风控规避：
- 严格按真实小程序的 UA / Referer 伪装
- 用真实抓包的请求格式（不是猜的）
- 不并发刷不同账号 / 不快速重抢

## 性能基准（真实测试）

- 启动到窗口可见：**< 1 秒**
- CK 验证：~500 ms
- 关键词搜索（"高达" 396 个结果）：**~150 ms**
- 完整下单链路（confirmOrder + createOrder）：**~1-2 秒**
- 内存占用：~250 MB（PyQt6 + Chromium 内核）

## 隐私 & 安全

- ✅ 所有数据**存本地**（`%APPDATA%\BandaiSniper\`）
- ✅ 软件**不联网上传**到任何第三方
- ✅ 只直连两个地方：万代服务器 + 你配的推送服务（如 Server 酱）
- ✅ 开源，源码在 GitHub 公开
- ⚠️ Windows Defender 偶有误报（PyInstaller 打包的 exe 通病），手动加信任
