## 13.5 UI 设计决策（**用户拍板**）

### 语言策略

**基础**：**中文界面**（所有按钮、提示、错误信息中文）

**商品名双语显示**：因为万代商品大多是日本产，在**中文名旁边显示日文原名**，帮助朋友确认是同一个商品。

示例：
```
预售 PB 万代模型 HGUC 1/144 里歇尔 (防卫者B型装备)
（日文原名：プレミアムバンダイ HGUC 1/144 リゲル ディフェンサー）
```

**实现**：
- 从 `spu/v2/detail/new` 响应里取 `nameCn`（中文名）
- **需要抓一下** `nameJp` 字段是否存在（HAR 里快速确认），有就直接显示
- 如无，做一个可选的"别名"字段让用户自己填

### 配色主题（阿姆罗高达 + 夏亚扎古混搭）

因为项目名叫 **RedComet**（夏亚"赤い彗星"），以夏亚扎古的红为主色调，点缀 RX-78-2 的白/蓝/黄。

| 色角色 | 色值 | 对应机体部位 | 用途 |
|---|---|---|---|
| `sazabi-red` 夏亚红（主） | `#C8122C` | 扎古主装甲 | 主按钮 / 倒计时大字 / 品牌色 |
| `char-pink` 夏亚粉 | `#E86A8C` | 扎古肩部 | 错误提示 / 次级强调（比夏亚红柔） |
| `dark-crimson` 暗红 | `#7A0A1C` | 扎古阴影 | 边框 / 按下态 |
| `gundam-white` 高达白 | `#F4F4F0` | RX-78-2 躯干 | 卡片 / 对话框背景 |
| `gundam-blue` 高达蓝 | `#113D6E` | 胸部肩甲 | 次级按钮 / 信息提示 |
| `gundam-yellow` 高达黄 | `#F7D200` | V 字天线 | 成功状态 / 关键指标高亮 |
| `gundam-red` 高达红 | `#D03329` | 脚部头盔 | 仅用于区别夏亚红的点缀（少量） |
| `ink` 墨 | `#1A1D2B` | —— | 主文字 / 深色背景模式 |
| `ash` 灰 | `#8A8D9B` | —— | 次要文字 / 分割线 |

**风格调**：
- 底色走 `gundam-white` 纸质感，不走黑色极客风
- 关键操作（开始抢购按钮）用 `sazabi-red` 满填
- 抢到订单时大字用 `gundam-yellow`（胜利 V 字感）
- 错误提示用 `char-pink`，柔和不刺眼
- 边角稍圆（8-12px），整体**干净、军武机甲感**，不走 iOS 毛玻璃

**字体**：
- 中文：系统默认（思源黑体 / 微软雅黑 fallback）
- 日文：Noto Sans JP（无就用 Meiryo / Yu Gothic fallback）
- 数字 / 倒计时：等宽 `JetBrains Mono`（没有就 `Consolas`）

## 13.6 依赖决策

| 用途 | 选择 | 理由 |
|---|---|---|
| GUI 容器 | `pywebview` | 比 Electron 轻，Python 原生集成 |
| 前端 JS | 原生（不上框架） | 逻辑不复杂，30-50 行 vanilla JS 够用；不引依赖减少打包体积 |
| 图标字体 | Material Symbols / Emoji | 不下字体包，用 emoji 也符合"日系"气质 |
| 打包 | `pyinstaller` | 生态成熟；onefile + windowed 模式 |
| 持久化 | Python 标准库 `json` + `sqlite3` | 无依赖 |
| 日期时间 | Python 标准库 + 前端原生 | 无依赖 |
