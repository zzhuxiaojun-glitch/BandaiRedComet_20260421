# 14. GUI 测试清单

**适用版本**：Phase 1 MVP + Phase 2.1 + Phase 2.2（commit `fc05cde` 之后）
**用途**：在 WSLg / Windows 上完整过一遍 GUI，把每个功能点都按一下，确认没有 regression。
**耗时**：20-30 分钟（不含真实抢购）

## 14.0 环境准备

**WSL 下首次跑 GUI 需要**：
```bash
# Qt 后端 + WebEngine
.venv/bin/pip install pyqt6 qtpy PyQt6-WebEngine

# CJK 字体（WSLg 预览看中文/日文不乱码）
sudo apt install -y fonts-noto-cjk
```

**每次启动命令**：
```bash
PYTHONPATH=src .venv/bin/python -m bandai_sniper gui --debug
```

`--debug` 会多开一个 "Web Inspector" 窗口（开发者工具），能看 console 报错。正式给朋友时不加。

## 14.12 已知限制 & 豁免项

不作为 bug 记录的预期行为：

- `[pywebview] GTK cannot be loaded` —— 我们用 Qt，不用 GTK，这行忽略
- `MESA: error: ZINK: failed to choose pdev` —— WSLg 图形驱动警告，不影响
- `Release of profile requested but WebEnginePage still not deleted` —— Qt 关闭时的清理 warning
- datetime-local 显示 `04/23/2026, 04:58 AM` —— 浏览器 locale 决定。Windows 中文系统会自动变 24 小时制
- 首次启动有 1-2 秒空白 —— pywebview 启动 Bottle HTTP server 和 Qt WebEngine 需要时间
