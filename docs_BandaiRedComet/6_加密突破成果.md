## 6.1 密钥与算法

解包 `wx1cb4557915b2b7cd/129/__APP__.wxapkg`（3.6 MB），在 `app-service.js` 中定位到配置模块 `EEE384F5E0022CCF8885ECF2CC2FACB7.js`：

```js
module.exports = {
    environment: "prod",
    mainApiDomain: "https://crm-app-api.bandainamcoshanghai.com/api/",
    ossUrl: "https://cdn-crm.bandainamcoshanghai.com/image/wx/",
    mtaAppId: "wx1cb4557915b2b7cd",
    secretKey: "5p/9OgW9F5cJc0fOs0TXYA==",       // AES-192 密钥（24 字节）
    publicKey: "EGs7AgSG7k3aj6dGnLnbPL3125OjI2Bq" // MD5 签名盐
};
```

**关键陷阱**：`secretKey` 看起来像 base64（以 `==` 结尾），**但它不是用于 base64 解码的**。源码里是 `CryptoJS.enc.Utf8.parse(secretKey)`，意思是把这个字符串**当成 UTF-8 字面量**，密钥的 24 个字节就是这 24 个 ASCII 字符 `5p/9OgW9F5cJc0fOs0TXYA==` 本身。

## 6.2 加密函数

源码（`app-service.js` line 10583）：

```js
aesEncrypt: function(e) {
    var t = r.default.enc.Utf8.parse(a);   // a = secretKey
    return r.default.AES.encrypt(e, t, {
        mode: r.default.mode.ECB,
        padding: r.default.pad.Pkcs7,
        keySize: 6   // 6 words = 24 bytes = AES-192
    }).toString();
}
```

Python 等价（`src/bandai_sniper/crypto.py`）：

```python
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad
from base64 import b64encode

SECRET_KEY = b"5p/9OgW9F5cJc0fOs0TXYA=="  # 24 字节

def aes_encrypt(plaintext: str) -> str:
    cipher = AES.new(SECRET_KEY, AES.MODE_ECB)
    return b64encode(cipher.encrypt(pad(plaintext.encode("utf-8"), 16))).decode()
```

## 6.3 签名函数

源码：

```js
signature = md5(userToken + sortedParamsForSign + nonce + timestamp + publicKey).toLowerCase()
```

`sortedParamsForSign` 的规则（源码 `o(e, r, t)`）：
- 按 key 字典序排序
- null / undefined 字段丢弃
- 嵌套对象用 `JSON.stringify` 打包成字符串（注意这一层是签名专用，加密时不做这步）
- 原始 array 包装成 `{"str": JSON.stringify(arr)}`

## 6.4 请求组装

```
GET  /api/xxx?encryptionUrlParams=<encodeURIComponent(aes(params))>
POST /api/xxx    body = {"encryptionBodyParams": "<aes(params)>"}
```

**所有接口共用 5 个 header**：

```http
api-access-token: <CK 长串>
timestamp:        <ms 时间戳>
nonce:            <0-999999 随机整数>
signature:        <md5 小写 32 字符>
trace_id:         <uuidv4>_<timestamp>[_<userId>]
Content-Type:     application/json
```
