## 7.4 核心踩坑索引

详见 [`9_踩坑汇总.md`](./9_踩坑汇总.md)，最贵的几个：

1. **JS md5 UTF-16 截断 bug**：`charCodeAt & 0xFF`，中文字符被废。Python 必须 `bytes(ord(c) & 0xFF for c in combo)`，不是 `.encode("utf-8")`
2. **JS `o(e, r, t)` 函数**：`t` 是 encryptionEnable（恒 true）**`r` 才是 sign/encrypt 开关**。顶层 Array 时 sign 包 `{"str": JSON.stringify(原数组)}`，inner JSON **不走 replacer**
3. **createOrder 入参** = confirmOrder 响应 data 字符串化回传（Python `str(142.0)` 错，要 `_js_str`）
4. **HTTP 432**：万代对缺 UA / Referer 的请求直接拒
5. **响应体整块就是 AES 密文**（无 JSON 外壳）
